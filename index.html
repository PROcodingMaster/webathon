    C language :

    It was mainly developed as a system programming language to write an operating system.
    The main features of C language include low-level access to memory, a simple set of keywords,
    and clean style, these features make C language suitable for system programmings
    like an operating system or compiler development.


    C++ language :

    C++ is a cross-platform language that can be used to create high-performance applications.
    It was developed by Bjarne Stroustrup at Bell labs in 1979, as an extension to the C language.
    It also gives programmers a high level of control over system resources and memory.
